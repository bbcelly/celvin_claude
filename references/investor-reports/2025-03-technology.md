# Investor Report March 2025 - Technology

*Due: 2025-04-04 | Completed*

- Media Library & Storefront API: The Media Library project is on hold due to capacity constraints, while the Storefront API team has analyzed performance, drafted a production architecture, and held strategic discussions on future development.
- WYSIWYG Editors & Frontend Enhancements: Both 3G and 4G editors saw UX improvements, code cleanups, and new features including session timeout handling, Storybook documentation for reusable components, and multilingual support.
- Testing & Quality Assurance: Expanded Cypress testing now covers functional, accessibility, and component tests, with performance metrics tracked via Grafana.
- Refactoring & Developer Productivity: Initiatives such as one-click project cloning, SQL optimizations, and adoption of coding best practices (e.g., @ts-check, Prettier) have streamlined development efforts.
- API & Developer Tools: A public preview of the new API SDK is now available, accompanied by significant progress in webhook siloization and a major overhaul of developer documentation.
- Logistics & Operations: A milestone of 1M shipments via Shoptet Baliky was reached, with new shipping services and backend improvements for handling delayed shipments and unified order references. Launched new Balikovna shipping services.
- Shoptet Pay (Internal + SC): After huge effort new projects are created on Adyen ABP. Gradual migration of existing projects will follow to achieve complete migration. Express checkout is already released for few projects in pilot. There are some edge cases we have to solve in upcoming weeks.
- Cash Register: Main part of refactoring of current solution is almost done. We are now much more familiar with it and understand it's functionality.
- Performance & Accessibility: Optimizations in image rendering, lazy loading, and accessibility features have notably boosted PageSpeed performance.
