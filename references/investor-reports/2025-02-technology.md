# Investor Report February 2025 - Technology

*Due: 2025-03-10 | Completed*

- Frontend performance improvements: We've shipped a huge INP & CLS metrics improvement leading to better UX and search engines ranking of our eshops.
- Jira to GitHub content migration: Work started on a tool that will allow us to migrate older content from Jira to GitHub Issues / Projects so that the teams can search and browser history within a single tool. This is one of the last stages of the overall migration away from Jira to a more modern toolchain based on GitHub.
- Apache to nginx migration: We're preparing a move away from the original web server, Apache, to a more performant and modern one called nginx. The new web server has been prepared in local development environment (devenv) and we're planning the rollout to both LAB and PROD environments in the coming months.
- Major auto-imports optimizations: The SRE team rolled out big improvements to our automatic imports, for example, it can save about 30% of the work/time on imports that are similar to this in the past, and many others.
- Logistics retryability: This concludes a multi-month effort to handle carrier outages better, and has been rolled out during February.
- Tech lecture in Product Academy: Pavel Kacha delivered a tech-oriented talk in the Product Academy series, describing advantages of our platform from the technology perspective.
- Media library project (prev. known as File Manager): more research done and CDN prototype completed.
- Cash Register refactoring: The Pay team is working on refactoring the current Cash Register to support one of the initiatives for 2025.
- Shoptet.cz migration to JAMStack: Final work on switching our public website to a new tech stack (React, Next.js, "JAMStack") was done in February. The final launch was in early March.
- Dibi upgrade: We're preparing Dibi (PHP database library) upgrade which is non-trivial; one of the staff engineers is working on it.
