## Goal

Achieve widespread AI-Enabled development across all Shoptet engineering teams, establishing an AI-First culture where AI coding assistants are the default development tool.

**6-month targets:** 98% daily usage, 50% AI code contribution, +60% productivity gain.

---

**Claude Code** — team.blue's primary recommendation. Validated by iubenda (also team.blue): 95% developer preference over Cursor, +67% productivity gain after mandatory training.

## The Non-Negotiable: Mandatory 2-Day Hands-Off Training

Per iubenda's finding — optional workshops fail (+18% vs +67%). Every cohort needs:

- **Day 1:** Concepts, installation, guided practice
- **Day 2:** Real backlog items, AI does the heavy lifting
- **No opt-outs.** Frame it as strategic imperative from CPTO.
- **Ratio:** 1 coach per 5-7 developers

## Setup Required Before Pilot Launch

**Measurement:**
- Baseline metrics for Offer team: sprint velocity, deployment frequency, lead time
- Git commit tagging convention: `ai-assisted` / `ai-generated`
- Usage dashboard (Claude Code session logs)

**Structure:**
- 1 AI champion per superdomain (identify enthusiasts, not volunteers)
- 1:5 coach-to-developer ratio for training events
- Monthly pulse survey on developer satisfaction

**Governance:**
- HoE accountability for team adoption rates
- Weekly usage review by superdomain lead
- 48-hour intervention protocol for non-users (Day 1: automated reminder, Day 2: coach outreach, Day 3: manager 1-on-1)
